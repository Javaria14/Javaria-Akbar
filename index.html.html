University Network System

Full campus network simulated in Cisco Packet Tracer — 3 departments, core router, WAN links, firewall, and DNS.

+

Architecture

  • Centralized Head Office (Lahore) connected to CS, HR, and Finance departments via WAN serial links
  • VLANs per department for traffic isolation — departments can't see each other's broadcast traffic
  • OSPF dynamic routing for automatic path calculation and fast convergence across all segments
  • VLSM (Variable Length Subnet Masking) for efficient IP address allocation per department size
  • Firewall at Head Office blocks unauthorized cross-department access
  • Centralized DNS server resolves hostnames across the entire university network
